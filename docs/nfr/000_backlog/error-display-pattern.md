# NFR: エラー表示パターンの定義

- Title: エラー表示パターンの定義
- Status: Idea
- Quality attributes (ISO/IEC 25010): Usability
- Created: 2026-01-15
- Last updated: 2026-01-15

## Goal
アプリケーション全体で一貫したエラー表示パターンを定義し、ユーザー体験を統一する。

## Non-goals
- 既存のトースト表示ライブラリの利用方針
- エラーログ（サーバーサイド）の仕様

## Acceptance criteria
- エラーの種類別（バリデーション、通信エラー、サーバーエラー等）に表示パターンが定義されている
- 表示パターンがコンポーネントまたはユーティリティとして実装可能な粒度で記述されている
